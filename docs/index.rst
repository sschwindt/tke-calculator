TKE Analyst
===========

This Python3 code aids in analyzing raw measurements with an Acoustic Doppler Velocimeter (ADV) producing ``*.vno`` and ``*.vna`` files.

.. important::

	``*.vno`` and ``*.vna`` files need to comply with the following name convetion:
    ``XX_YY_ZZ_something.vna``
